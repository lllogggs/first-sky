# 첫 하늘

출생일, 출생시간, 태어난 곳을 입력하면 태어난 순간의 별자리와 과거 날씨를 결합해 9:16 공유용 스토리 결과를 보여주는 정적 웹 앱입니다.

## 핵심 흐름

- 진입 화면: 입력 폼만 표시
- 결과 화면: 별도 버튼 없이 9:16 스토리로 전환
- 스토리 연출: 하늘이 먼저 열리고, 타입명과 해석이 순차 등장
- 공유 콘텐츠: 태양, 달, 상승 별자리 기반의 세 꼭지 해석 중심
- 영상 저장: 브라우저 MediaRecorder 기반 세로 영상 다운로드

## 데이터

- 장소 검색: Open-Meteo Geocoding API
- 과거 날씨: Open-Meteo Historical Weather API
- 점성술 계산: 브라우저 내부 근사 계산

Historical Weather API는 1940년 이후 데이터를 제공합니다. 더 오래된 출생일은 날씨 기록 없음 상태로 처리됩니다.
